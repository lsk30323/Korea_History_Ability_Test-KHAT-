## 3. 자세한 설명 (최대 4000자)
```
공무원 시험과 한국사능력검정시험(한능검)을 함께 대비하는 한국사 문제집 앱입니다.
시대별로 정리된 예상문제를 풀고, 바로 채점하고, 틀린 문제는 오답노트로 모아 복습할 수 있습니다.

■ 이런 분께 좋아요
· 공무원 시험의 한국사 과목을 준비하는 수험생
· 한국사능력검정시험(심화) 합격을 목표로 하는 분
· 자투리 시간에 시대별로 문제를 풀며 약점을 보완하고 싶은 분

■ 주요 기능
· 시대별 학습 — 선사·고대·고려·조선 전기/후기·근대·일제강점기·현대 등 13개 영역으로 분류
· 즉시 채점과 해설 — 정답 확인과 함께 핵심 포인트를 한 줄로 정리
· 오답노트 — 틀린 문항을 자동으로 모아 다시 풀기, CSV 내보내기 지원
· 과목별 정답률 통계 — 약한 시대부터 집중 학습
· 회차/과목/문항 수 선택, 문항 순서 섞기
· 오프라인 동작 — 설치 후 인터넷 없이 어디서나 학습

■ 수록 문항
· 시대별 자체 제작 예상문제와 모의고사를 폭넓게 수록
· 기출 원문은 수록하지 않으며, 표준 교육과정에 기반한 자체 제작 문항으로 구성

■ 개인정보 보호
· 어떠한 개인정보도 수집하지 않습니다.
· 학습 기록(오답노트·통계·진행 상황)은 사용자 기기에만 저장되며 외부로 전송되지 않습니다.
· 광고, 사용자 추적, 제3자 분석 도구를 사용하지 않습니다.

본 앱은 학습 보조용이며, 시험 시행 기관과는 관련이 없습니다.
```

## 4. 데이터 보안(Data safety) 양식 답변

> 핵심: 이 앱은 **데이터를 수집·전송·공유하지 않습니다.** 모든 학습 기록은 기기 내
> 로컬 저장소에만 보관됩니다(= Google 기준 ‘수집’에 해당하지 않음).

| 질문 | 답변 |
|---|---|
| 앱이 사용자 데이터를 **수집 또는 공유**하나요? | **아니요 (No)** |
| 수집하는 데이터 유형(위치/개인정보/금융 등) | **없음** (모두 선택 안 함) |
| 데이터를 제3자와 **공유**하나요? | **아니요** |
| 전송 중 데이터 **암호화**? | 해당 없음(전송되는 데이터 없음) |
| 사용자가 데이터 **삭제를 요청**할 수 있나요? | 기기 내 저장만 하므로, 앱 내 ‘초기화’ 또는 앱 삭제로 즉시 삭제 가능 |
| 광고 ID 사용 | **아니요** |

- 앱 콘텐츠 광고 포함 여부: **광고 없음 (No ads)**
