# 공무원 한능검 문제집 (개인용)

기존 프로젝트(`docs/` PWA, `study-materials/`, Capacitor 앱)와 **분리된 개인 학습용 앱**입니다.
풀기 · 채점 · 과목별 분류 · 오답노트 기능을 제공합니다.

## 수록 문항

- **모의고사 1~20회 = 1,000문항** (`../docs/data/exam-1~20.js`를 그대로 참조)
- 자체제작 샘플 6문항(`data/bank.js`)
- 13개 과목 분류: 선사·고대·고려·조선전기·조선후기·근대·일제강점기·현대·세시풍속·유네스코·지역사·인물사·문화재

> 모의고사 데이터는 `docs/data`의 **단일 원본**을 참조합니다(중복 저장 없음).
> `docs/data`를 갱신하면 이 앱도 자동으로 반영됩니다.
> `data/adapter.js`가 `EXAM_DATA` → `QUESTION_BANK` 스키마 변환을 담당합니다.

## 실행

별도 빌드 없이 브라우저에서 바로 열면 됩니다.

```
gongmuwon-app/index.html  →  더블클릭(파일 열기) 또는
python3 -m http.server     →  http://localhost:8000/gongmuwon-app/
```

> `../docs/data` 를 참조하므로, http 서버는 **저장소 루트**에서 실행하세요.
> 파일을 직접 열 때도 `gongmuwon-app` 위치 기준 상대경로(`../docs`)로 동작합니다.

진행상황과 오답노트는 브라우저 `localStorage`에 저장됩니다(이 기기/브라우저 한정).

## 기능

| 기능 | 설명 |
|---|---|
| 과목 선택 | `area`별로 골라 풀기 (미선택 시 전체) |
| 문항 수 | 10/20/50/전체 |
| 순서 섞기 | 출제 순서 랜덤화 |
| 즉시 채점 | 보기 선택 시 정답·해설·오답 사유 표시 |
| 오답노트 | 틀린 문항 자동 저장 → "오답노트 문항만 풀기"로 복습 |
| 이어풀기 | 풀던 세션 자동 복원 |

## 문항 추가 — `data/bank.js`

`window.QUESTION_BANK` 배열에 객체를 추가합니다. 스키마는 `docs/data/exam-*.js`와 호환됩니다.

```js
{
  "id": "고대-001",            // 고유 ID (오답노트 식별용)
  "area": "고대",              // 과목(분류). 홈 화면 칩이 자동 생성됨
  "source": "예상",            // 샘플 | 예상 | 업로드
  "origin": "자체제작",        // 세부 출처 표기(선택)
  "passage": "사료/제시문…",   // 없으면 "" 또는 생략
  "q": "발문…?",
  "options": ["①내용","②","③","④","⑤"],
  "answer": 2,                 // 정답 보기 번호(1부터 시작)
  "reason": "정답 해설…",
  "wrong": { "1": "오답 사유", "3": "…" },  // 선택
  "keyword": "한 줄 핵심"      // 선택
}
```

### `source` 표기 규칙 (중요)
- **샘플** — 앱 동작 확인용 자체제작 예시
- **예상** — 기출 *유형* 기반 자체제작 예상문제 (기출 원문 아님)
- **업로드** — 사용자가 합법적으로 보유해 제공한 자료를 가공한 문항

> ⚠️ 한국사능력검정시험 **기출 원문은 국사편찬위원회의 저작물**입니다.
> 원문을 그대로 복제해 넣지 않으며, 위 출처 구분에 맞는 문항만 수록합니다.
> 본 앱은 개인 학습 용도이며 배포(스토어 등록) 대상이 아닙니다.

## 대량 추가 팁
- 한 과목 파일을 따로 두고 싶다면 `data/bank-고대.js` 처럼 나눈 뒤
  `index.html`에 `<script src="./data/bank-고대.js"></script>`를 추가하고,
  각 파일에서 `window.QUESTION_BANK = (window.QUESTION_BANK||[]).concat([... ])` 형태로 이어 붙이면 됩니다.
