# 한능검 2급 오답노트 템플릿

> 모의고사·기출을 풀고 **틀린 문항 + 찍어서 맞은 문항**을 여기에 기록합니다. 핵심은 "왜 틀렸는지 유형을 분류"하고 "혼동한 짝을 박제"하는 것.

## 사용법
1. 채점 후 틀린 문항을 아래 **오답 기록 블록**에 복사해 채운다.
2. **틀린 유형**을 분류해 내 약점 패턴(개념? 혼동? 순서?)을 파악한다.
3. **D-7 / D-3 / D-1** 세 번 재복습 체크. 세 번 다 맞으면 졸업(✓✓✓).
4. 맨 아래 **누적 통계**로 약점 영역과 반복 함정을 추적해 복습 우선순위를 정한다.

## 틀린 유형 분류표
| 코드 | 유형 | 처방 |
|---|---|---|
| ① | 개념 자체를 몰랐다 | 워크북 개념이해부터 다시 |
| ② | 비슷한 개념을 혼동했다 | 혼동 짝을 묶어 암기 |
| ③ | 자료(사료)를 잘못 읽었다 | 자료 속 핵심 단서(키워드) 찾는 연습 |
| ④ | 순서·연표를 헷갈렸다 | 必出 순서 6종 반복 |
| ⑤ | 아는데 실수했다 | 시간 배분·마킹 점검 |

## ✍️ 작성 예시 (이렇게 채우세요)

### [ 3회 8번 ] 영역: 고대
- **문제 핵심:** 신문왕의 업적으로 옳은 것은?
- **내 답 → 정답:** ( 정전 지급 ) → ( 관료전 지급·녹읍 폐지 )
- **틀린 유형:** ② 혼동
- **왜 틀렸나 / 정답 근거:** 정전=성덕왕, 녹읍 부활=경덕왕. **신문왕은 관료전 지급 + 녹읍 폐지(왕권 강화)**.
- **혼동한 짝:** 관료전(신문왕) ↔ 정전(성덕왕) ↔ 녹읍부활(경덕왕)
- **재복습:** D-7 ☑  D-3 ☐  D-1 ☐

### [ 1회 24번 ] 영역: 근대
- **문제 핵심:** 단발령이 시행된 개혁은?
- **내 답 → 정답:** ( 갑오개혁 ) → ( 을미개혁 )
- **틀린 유형:** ② 혼동
- **왜 틀렸나 / 정답 근거:** 신분제·과거제 폐지=1차 갑오개혁(군국기무처) / **단발령·태양력=을미개혁**.
- **혼동한 짝:** 갑오개혁 ↔ 을미개혁
- **재복습:** D-7 ☑  D-3 ☑  D-1 ☐
