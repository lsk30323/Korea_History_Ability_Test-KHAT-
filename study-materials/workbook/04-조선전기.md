# 실전 워크북 ④ 조선 전기 (건국 ~ 임진왜란 이전)

> 한국사능력검정시험 **심화(2급 목표)** 대비 워크북.
> 대상 구간: 1392 태조 즉위 ~ 1592 임진왜란 직전.
> 구성: ① 개념이해 → ② 실전문제 6문항 → ③ 정답 및 해설.

---

## 1. 개념이해

조선 전기는 **"세종 + 4대 사화 + 경국대전·3사 + 토지제도"** 네 묶음이 점수의 대부분을 만든다. 흐름과 인과를 잡으면 보기 지문이 거의 그대로 재출제되는 이 구간에서 안정적으로 고득점할 수 있다.

### (1) 왕별 업적과 인과 — 통치 체제는 "왕권 ↔ 신권"의 진자운동

조선 전기 통치 제도의 핵심 줄기는 **왕권 강화와 재상(신권) 위임 사이를 오가는 흐름**이다. 이 진자운동을 이해하면 왕별 업적이 따로 노는 암기가 아니라 하나의 이야기로 꿰어진다.

| 왕 | 핵심 업적 (★=단골) | 권력 방향 |
|---|---|---|
| **태조** | 조선 건국(1392), 한양 천도, **정도전**(재상 중심 정치, 『조선경국전』, 불교 비판 『불씨잡변』) | 재상 중심 |
| **태종** | ★**6조 직계제**(6조가 의정부를 거치지 않고 왕에게 직접 보고 → 왕권 강화), **호패법**, 사병 혁파, 신문고, 양전·사원전 정리 | 왕권 ↑ |
| **세종** | ★**집현전**, ★**훈민정음** 창제, **의정부 서사제**(재상에게 권한 위임), **칠정산**(역법), 측우기·앙부일구·자격루, 『농사직설』, **4군 6진**(최윤덕·김종서, 압록강~두만강), 쓰시마 정벌(이종무), 3포 개항 | 재상권 ↑ |
| **세조**(수양대군) | 계유정난으로 집권, ★**6조 직계제 부활**, **직전법**(현직 관리에게만 수조권), 집현전·경연 폐지, 『경국대전』 편찬 **시작** | 왕권 ↑ |
| **성종** | ★『경국대전』 **완성·반포**(통치 규범 확립), ★**홍문관** 설치(집현전 계승, 경연 주관), 『동국통감』·『동국여지승람』·『국조오례의』·『악학궤범』 편찬 | 균형·완성 |

**인과로 읽기**
- 태종이 **6조 직계제**로 왕권을 세웠고 → 세종은 안정된 왕권 위에서 **의정부 서사제**로 재상에게 권한을 위임하며 집현전 학자들과 문화를 꽃피웠다.
- 세조는 계유정난으로 집권한 뒤 다시 **6조 직계제를 부활**시켜 왕권을 강화하고, 집현전·경연을 폐지했다. 동시에 토지 부족을 풀기 위해 **직전법**(현직 관리에게만 수조권)을 시행하고 **『경국대전』 편찬을 시작**했다.
- 성종은 세조가 시작한 **『경국대전』을 완성·반포**하여 통치 규범을 확립하고, 세조가 없앤 집현전의 학문 기능을 **홍문관**으로 되살렸다.

> **★ 최대 단골 함정**: 『경국대전』은 **세조 때 "편찬 시작" → 성종 때 "완성·반포"**. 시험은 "완성 = 성종"을 묻는다. "세조가 완성"으로 낚는 보기에 주의.

### (2) 통치 체제 — 중앙·지방·3사

- **중앙**: **의정부**(재상 합의·국정 총괄) + **6조**(이·호·예·병·형·공, 실무 집행).
- **3사(언론 기구)** — 권력 독점을 견제하는 핵심:

| 기구 | 기능 |
|---|---|
| **사헌부** | 감찰 |
| **사간원** | 간쟁·봉박 |
| **홍문관** | 경연·학문·정책 자문 (집현전 계승, 성종 설치) |

  → 사헌부 + 사간원 = **양사(대간)**, 서경권 행사.
- **지방**: 전국 **8도** → 부·목·군·현. **모든 군현에 수령(지방관) 파견**(고려와의 핵심 차이), 향리는 수령 보좌로 지위 격하. 유향소(향촌 자치)·경재소.
- **군사**: 5위(중앙), 진관 체제(지방).
- **과거**: 문과(대과)·무과·잡과, 문과 중시. 승과는 점차 폐지. 음서·천거 보조.

> **계보 혼동 금지**: **집현전(세종) → 홍문관(성종)** 은 학문·언론 기구, **비변사**(여진·왜구 대비 임시기구 → 임진왜란 후 상설·국정 최고 기구)는 군사·국정 총괄. 셋을 섞지 말 것.

### (3) 사림 vs 훈구, 4대 사화 — 순서가 생명

- **훈구**: 세조 즉위 공신 계열, 기득권 세력.
- **사림**: 지방 중소지주·성리학 명분 중시. **성종 때 김종직 등용**으로 중앙 진출(주로 3사 언관직).
- 두 세력의 충돌이 곧 **4대 사화**다.

| 사화 | 연도 | 왕 | 핵심 원인/사건 |
|---|---|---|---|
| **무오사화** | 1498 | 연산군 | **김종직의 「조의제문」**(사초 문제) → 훈구가 사림 공격 |
| **갑자사화** | 1504 | 연산군 | **폐비 윤씨 사사 사건** 보복 |
| **기묘사화** | 1519 | 중종 | ★**조광조**의 개혁(현량과·위훈 삭제·소격서 폐지) 반발 → "주초위왕(走肖爲王)" 모함 |
| **을사사화** | 1545 | 명종 | 외척 **대윤(윤임) vs 소윤(윤원형)** 권력 다툼 |

> **두문자 암기**: **무–갑–기–을 / 연–연–중–명** (연산군 두 번 → 중종 → 명종).

- 사화로 타격을 입은 사림은 **향촌으로 낙향 → 서원·향약으로 세력 재건 → 선조 때 정계 장악**(붕당 정치로 연결).
  - **서원**: 선현 제사 + 후학 교육. 최초 **백운동 서원**(주세붕) → 이황 건의로 **소수서원** 사액.
  - **향약**: 향촌 자치 규약. 4대 덕목 = **덕업상권·과실상규·예속상교·환난상휼**. 풍속 교화·사림의 농민 통제 기반.

### (4) 대외 관계 — 사대교린

- **사대(명)**: 조공·책봉 형식의 친선·실리 외교(자주성 유지).
- **교린(여진·일본)**: 회유(무역소·3포)와 강경(정벌)을 병행.
  - **여진**: **4군 6진** 개척(세종, 최윤덕·김종서, 영토를 압록강~두만강까지 확장), 사민·토관 정책.
  - **일본**: 세종 때 **쓰시마(대마도) 정벌**(이종무) → **3포 개항**(부산포·제포·염포) → 계해약조로 제한 무역. 이후 삼포왜란·을묘왜변 → **비변사** 설치 계기.

### (5) 문화·사상

- **성리학**의 국가 통치 이념화. **이황**(『성학십도』, 주리론, 영남학파) · **이이**(『성학집요』, 주기론, 기호학파) — 단, 본격 학문 논쟁은 16세기.
- **과학·편찬**: 세종대 **칠정산·『농사직설』·측우기·자격루**, 성종대 **『동국통감』(역사)·『동국여지승람』(지리)·『국조오례의』(의례)·『악학궤범』(음악)**, 태조대 **혼일강리역대국도지도**(현존 동양 최고 세계지도).
- **공예**: **분청사기(15세기) → 순백자(16세기)** 로 변화. (청화백자는 조선 후기 유행.)
- **인쇄**: 활자(계미자·갑인자) 개량.

## 3. 정답 및 해설

### 문제 1 — 정답 ③ 세종

- **근거**: 집현전 설치, 칠정산 편찬, 최윤덕·김종서를 통한 4군 6진 개척은 모두 **세종**의 대표 업적이다.
- **오답 반박**
  - ① 태조 — 조선 건국·한양 천도·정도전 등용이 핵심. 집현전·칠정산과 무관.
  - ④ 세조 — 6조 직계제 부활·직전법·『경국대전』 편찬 시작이 핵심. 4군 6진은 세종.
  - ⑤ 성종 — 『경국대전』 완성·홍문관 설치가 핵심.
- **키워드**: 집현전·칠정산·4군 6진 = **세종**.

### 문제 2 — 정답 ② 신문고를 처음 설치하였다

- **근거**: 자료의 **6조 직계제**(6조가 의정부를 거치지 않고 왕에게 직접 보고)와 **호패법·사병 혁파**는 **태종**의 정책이다. 태종은 **신문고**도 설치하였다.
- **오답 반박**
  - ① 훈민정음 창제 → 세종.
  - ③ 홍문관 설치 → 성종.
  - ④ 『경국대전』 완성·반포 → 성종.
  - ⑤ 직전법 시행 → 세조.
- **키워드**: 6조 직계제·호패법 = **태종** → 신문고.

### 문제 3 — 정답 ② (나) - (가) - (라) - (다)

- **근거**: 4대 사화의 순서는 **무오(1498) → 갑자(1504) → 기묘(1519) → 을사(1545)**.
  - (나) 무오(「조의제문」·사초) → (가) 갑자(폐비 윤씨 사사) → (라) 기묘(조광조·주초위왕) → (다) 을사(대윤 vs 소윤).
- **오답 반박**
  - ①·④ 갑자 또는 기묘를 맨 앞에 두어 무오보다 앞세운 오류.
  - ③ 기묘(라)와 갑자(가)의 순서를 뒤바꿈 — 갑자(1504)가 기묘(1519)보다 앞.
  - ⑤ 을사(다)와 기묘(라)의 순서를 뒤바꿈 — 기묘(1519)가 을사(1545)보다 앞.
- **키워드**: **무–갑–기–을 / 연–연–중–명**.

### 문제 4 — 정답 ③ 홍문관

- **근거**: 성종 때 설치, **집현전의 학문 기능 계승**, 경연 주관·정책 자문, 사헌부·사간원과 함께 3사 구성 → **홍문관**.
- **오답 반박**
  - ① 의정부 — 재상 합의·국정 총괄 기구. 3사가 아님.
  - ④ 비변사 — 여진·왜구 대비 군사 기구로 출발해 임진왜란 후 국정 최고 기구화. 학문 기능·3사와 무관. (집현전 → 홍문관 계보와 혼동 주의.)
  - ⑤ 춘추관 — 역사 편찬 담당. 3사가 아님.
- **키워드**: 집현전 계승 + 3사 = **홍문관(성종)**.

### 문제 5 — 정답 ② 세조 때 편찬을 시작하여 성종 때 완성·반포되었다

- **근거**: 『경국대전』은 **세조 때 편찬 시작 → 성종 때 완성·반포**된 조선의 기본 법전이다.
- **오답 반박**
  - ① 정도전의 『조선경국전』은 태조 때의 별개 사찬 법전. 『경국대전』과 다름.
  - ③ 세종이 집현전과 완성했다는 설명은 사실이 아님 — 완성은 성종.
  - ⑤ **세조가 "완성·반포"했다는 최대 단골 함정**. 세조는 '시작'일 뿐, 완성은 성종.
- **키워드**: 시작 = **세조**, 완성·반포 = **성종**.

### 문제 6 — 정답 ④ 의정부 서사제를 실시하여 재상에게 권한을 위임하였다

- **근거**: **의정부 서사제는 세종**의 정책이다. 세조는 오히려 **6조 직계제를 부활**시켜 왕권을 강화했으므로, ④는 세조의 정책으로 옳지 않다.
- **오답(= 세조의 옳은 정책) 확인**
  - ① 계유정난으로 집권 — 맞음.
  - ② 6조 직계제 부활 — 맞음.
  - ③ 직전법(현직 관리에게만 수조권) 시행 — 맞음.
  - ⑤ 집현전·경연 폐지 — 맞음.
- **키워드**: 의정부 서사제(세종) ↔ 6조 직계제 부활(세조) — 시행 왕과 권력 방향을 반대로 적는 함정 주의.
