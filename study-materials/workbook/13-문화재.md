# 워크북 ⑬ 문화재 — 탑·불상·건축·자기 — 실전 학습

> 한능검 **2급(심화) 대비**. 문화재는 평균 3~5문항으로 비중이 큰 핵심 블록이면서, **내용은 알아도 양식(모양)을 몰라 틀리는 영역 1순위**다. 심화 문제지는 전면 컬러라 사진을 던지고 시대를 묻지만, 이 워크북은 사진 대신 **양식 설명(지문)**으로 같은 능력을 훈련한다. 핵심은 단일 암기가 아니라 **"이 시대 = 이 양식" 매칭**이다.

---

## 1. 개념이해 — 외우지 말고 "시대 = 양식"으로 묶는다

문화재가 어려운 게 아니라, **같은 형태(10층 대리석 탑, 거대 불상)를 다른 시대로 바꿔 내는 함정**이 어렵다. 시험이 노리는 건 "경천사지를 조선이라 한다", "관촉사를 통일신라라 한다" 같은 한 칸 어긋남이다. 그래서 단순 나열이 아니라 **무엇을 무엇과 헷갈리게 하는가**를 기준으로, 양식 키워드를 시대와 1:1로 못 박는다.

### (1) 탑(塔) — 시대별 양식 구분표 ★최빈출★

탑은 사진(지문) 단서로 시대를 가르는 게 핵심이다. **재료·층수·평면**이 곧 시대 단서다.

| 시대 | 대표 탑 | 재료·양식 | **시대 맞히는 단서 키워드** |
|---|---|---|---|
| **백제** | 익산 **미륵사지 석탑** | 화강암 석탑 | 현존 **最古·最大** 석탑. **목탑 양식을 돌로** 번안. 일부만 남음 |
| **백제** | 부여 **정림사지 5층석탑** | 화강암 석탑 | **날렵·균형**의 백제 미학. '평제탑'(소정방 글씨)으로도 불림 |
| **신라(삼국)** | 경주 **분황사 모전석탑** | **모전석탑**(벽돌 모양 돌) | 현존 신라 **最古** 석탑. **벽돌처럼 다듬은 돌**이 결정적 단서 |
| **신라(삼국)** | **황룡사 9층 목탑** | **목탑** | 선덕여왕·자장 건의. **몽골 침입 때 소실**(현존 X) |
| **통일신라** | 불국사 **3층석탑(석가탑)** | 화강암, **이중기단 3층** | 신라 석탑의 **완성형·전형**. 무구정광대다라니경 발견 |
| **통일신라** | 불국사 **다보탑** | 화강암, **이형(異形)** | **화려·복잡**한 비대칭. 석가탑과 동서로 나란히 |
| **고려** | 평창 **월정사 8각9층석탑** | **다각다층탑** | 고려 전기. **8각·여러 층** = 고려 단서. 송 영향 |
| **고려** | 개성 **경천사지 10층석탑** | **대리석**, 10층 | **1348년, 원(元) 양식** 도입. 흔치 않은 **대리석** |
| **조선(초)** | 서울 **원각사지 10층석탑** | **대리석**, 10층 | 세조(**1467**). **경천사지를 그대로 계승**한 쌍둥이 탑 |

> **흐름 한 줄:** 백제(목탑→석탑 번안, 단정) → 신라(모전석탑·목탑) → 통일신라(석가탑 = 이중기단 3층 정형 완성) → 고려(다각다층·대리석) → 조선(원각사지, 대리석 계승).

**사진 단서 3종 즉답** — 시험장에서 이 셋이면 절반은 푼다.
- **벽돌 모양 돌(모전석탑)** → **신라(삼국)** (분황사)
- **8각·여러 층 / 대리석 10층** → **고려** (월정사 / 경천사지)
- **이중기단 3층의 단정한 정형** → **통일신라** (석가탑)

### (2) 불상(佛像) — 시대별 양식

| 시대 | 대표 불상 | 시대 맞히는 단서 |
|---|---|---|
| **삼국** | **금동미륵보살반가사유상** | 한쪽 다리를 무릎 위에, **손가락을 뺨**에 댄 사유(思惟) 자세. 일본 고류사 목조반가사유상에 영향 |
| **통일신라** | **석굴암 본존불** | 인공 석굴, **사실적·이상적 비례의 절정** |
| **고려** | 논산 관촉사 **석조미륵보살입상**(은진미륵) | **거대·토속적·비례 파격**. 머리에 **네모난 갓(보개)**. 지방 호족기 대형 불상 |
| **신라말~고려초** | **철불**(예: 하남 하사창동 철조석가여래) | **철(鐵)** 재질. 호족 세력 성장 배경 |

> **흐름 한 줄:** 삼국(반가사유상, 섬세) → 통일신라(석굴암, 완벽한 사실미) → 고려(관촉사·철불, 거대·자유분방·지방색).

**불상 함정 한 줄 정리:** "완벽한 비례·사실미"면 **통일신라(석굴암)**, "거대하고 균형이 안 맞으며 토속적(네모난 갓)"이면 **고려(관촉사)**. 양식이 정반대라 바꿔 내기 쉽다.

### (3) 목조 건축(고려) — 주심포 양식 3대 천왕

- **부석사 무량수전 / 수덕사 대웅전 / 봉정사 극락전** = 고려 **주심포** 양식 대표.
  - **주심포:** 공포(栱包)를 **기둥 위에만** 둠 → 간결·단정. (vs 조선 **다포**: 기둥 사이에도 공포 → 화려)
  - **부석사 무량수전:** 배흘림기둥, 의상이 창건한 부석사. 고려 건축의 대표.
  - **봉정사 극락전:** 현존 **最古 목조 건축**으로 흔히 언급.
- **조선:** 궁궐(경복궁·창덕궁 등) + **다포** 양식 확대.

### (4) 자기(磁器) 흐름 ★시대순 단골★

> **고려청자 → (조선 초)분청사기 → 백자** — 이 순서가 곧 문제다.

| 단계 | 시대 | 특징 단서 |
|---|---|---|
| **고려청자** | 고려 | 비색(翡色). 12C 후반~13C **상감청자** 절정 → **무신정권기** 전성, 몽골 간섭기 쇠퇴. 강진·부안이 대표 생산지 |
| **분청사기** | **조선 초(과도기)** | 고려청자→백자 사이 **과도기**. 청자에 분장(흰 흙) → 소박·실용 |
| **조선 백자** | 조선 | 검소·실용 숭상(성리학)으로 주류화. 순백자 → 후기 **청화백자**(푸른 그림) |

> **자기는 순서만 확실히:** 청자(고려) → 분청사기(조선 초) → 백자(조선). 분청사기가 "고려와 조선 백자 사이 과도기"라는 위치를 못 박으면 순서 문제는 끝난다.

### (5) 인쇄·기록 문화 (자주 끼어드는 선지)

| 유물 | 시대 | 단서 |
|---|---|---|
| **무구정광대다라니경** | **통일신라** | **석가탑**에서 발견, 현존 **最古 목판 인쇄물** |
| **팔만대장경(재조대장경)** | **고려** | **몽골 격퇴 염원**, 합천 해인사 장경판전 보관 |
| **직지심체요절** | **고려 말** | 현존 **最古 금속활자본**(1377), 청주 흥덕사. 현재 프랑스 소장 |

> **순서:** 무구정광대다라니경(통일신라) → 팔만대장경(고려) → 직지심체요절(고려 말).

## 2. 실전문제 (5문항)

> 심화 실제 시험은 사진을 제시하지만, 여기서는 **양식 설명(지문)**으로 대체한다. 사진 자리에 들어갈 시각 단서를 글로 묘사했으니, "이 설명이 어느 시대·어느 문화재인가"를 그려 가며 푼다.

① 익산 미륵사지 석탑
② 경주 분황사 모전석탑
③ 평창 월정사 8각 9층 석탑
④ 개성 경천사지 10층 석탑
⑤ 서울 원각사지 10층 석탑

## 3. 정답 및 해설

### 문제 1 — 정답 **②**

- **자료 식별:** **벽돌 모양으로 다듬은 돌** + **신라 최고(最古) 석탑** → **경주 분황사 모전석탑(신라, 삼국)**.
- **정답 근거:** 모전석탑은 "벽돌처럼 다듬은 돌"이 결정적 단서이고, 현존 신라 최고 석탑이므로 ②.
- **오답 풀이(=다른 시대 양식):**
  - ① 익산 미륵사지 석탑 → **백제**. 화강암 석탑으로, 목탑 양식을 돌로 번안한 현존 最古·最大 석탑(모전석탑 아님).
  - ③ 월정사 8각 9층 석탑 → **고려**. 8각·다층의 다각다층탑.
  - ④ 경천사지 10층 석탑 → **고려**(대리석, 1348, 원 양식).
  - ⑤ 원각사지 10층 석탑 → **조선**(세조, 1467, 대리석).
- **한 줄 키워드:** 벽돌 모양 돌 = 모전석탑 = **신라(분황사)**.

### 문제 2 — 정답 **③**

- **자료 식별:** **대리석 10층** + **1348년** + **원 양식** → **개성 경천사지 10층 석탑(고려)**.
- **정답 근거:** 흔치 않은 대리석에 10층, 1348년 원 양식 도입이라는 단서가 모두 경천사지를 가리키므로 ③.
- **오답 풀이:**
  - ④ **원각사지 10층 석탑** → **조선 세조(1467)**. 둘 다 대리석 10층이라 사진이 비슷하지만, **경천사지(고려) → 원각사지(조선)** 순으로 시대가 다르다. 원각사지는 경천사지를 **계승**한 쌍둥이 탑이라는 점이 함정의 핵심. ★최대 혼동 함정★
  - ① 정림사지 5층 석탑 → **백제**(화강암, 날렵·균형).
  - ② 석가탑 → **통일신라**(화강암 이중기단 3층).
  - ⑤ 미륵사지 석탑 → **백제**(화강암, 목탑 양식 번안).
- **한 줄 키워드:** 대리석 10층 + 1348 + 원 양식 = **경천사지(고려)**. (1467·계승 = 원각사지·조선!)

### 문제 3 — 정답 **④**

- **자료 식별:** **거대·토속적·비례 파격 + 네모난 갓(보개) + 지방 호족기** → **관촉사 석조미륵보살입상(은진미륵) = 고려**. 따라서 **고려의 문화유산**을 고르는 문제.
- **정답 근거:** 보기 중 고려 문화유산은 **④ 월정사 8각 9층 석탑**(고려 전기 다각다층탑)이므로 ④.
- **오답 풀이(=다른 시대):**
  - ③ **석굴암 본존불** → **통일신라**. 사실적·이상적 비례의 절정으로, 거대·파격의 관촉사와 양식이 **정반대**다. 시대(고려)를 묻는데 통일신라 대표 불상을 끼워 넣은 ★혼동 함정★.
  - ① 분황사 모전석탑 → **신라(삼국)**. ② 정림사지 5층 석탑 → **백제**.
  - ⑤ 금동미륵보살반가사유상 → **삼국**(손가락을 뺨에 댄 사유 자세).
- **한 줄 키워드:** 거대·네모난 갓·호족기 = 관촉사 = **고려** → 같은 고려 = **월정사**. (완벽한 비례·사실미 = 석굴암·통일신라!)

### 문제 4 — 정답 **①**

- **자료 식별:** **통일신라** 문화유산만 고르는 문제.
- **정답 근거:**
  - ㄱ. 불국사 3층 석탑(석가탑) → **통일신라**(이중기단 3층 정형, 무구정광대다라니경 발견). ○
  - ㄴ. 석굴암 본존불 → **통일신라**(사실적·이상적 비례의 절정). ○
  - 따라서 통일신라끼리 묶은 **ㄱ, ㄴ = ①**.
- **오답 풀이(=다른 시대):**
  - ㄷ. 관촉사 석조미륵보살입상 → **고려**(거대·토속·네모난 갓). 석굴암(통일신라)과 바꿔 내는 함정.
  - ㄹ. 경천사지 10층 석탑 → **고려**(대리석, 1348, 원 양식).
- **한 줄 키워드:** 석가탑 + 석굴암 = **통일신라**. (관촉사·경천사지 = 고려!)

### 문제 5 — 정답 **②**

- **자료 식별:** 자기의 시대 흐름 → **고려청자 → 분청사기 → 백자**.
- **정답 근거:** (나) 고려청자(상감청자, 고려) → (가) 분청사기(조선 초 과도기) → (다) 조선 백자 순이므로 **(나)-(가)-(다) = ②**.
- **오답 풀이:** 분청사기는 **고려청자와 조선 백자 사이의 과도기**다. (가)분청사기를 맨 앞에 둔 ①, 백자를 앞에 둔 ④·⑤, 백자와 분청사기 순서를 뒤집은 ③은 모두 이 과도기 위치를 어긴 배열이다.
- **한 줄 키워드:** 청자(고려) → 분청사기(조선 초) → 백자(조선).

### ⚠️ 시험장 직전 4대 함정 체크

1. **경천사지(고려, 1348, 원 양식) ↔ 원각사지(조선 세조, 1467, 계승)** — 둘 다 대리석 10층 쌍둥이지만 시대가 다르다.
2. **석굴암(통일신라, 사실미) ↔ 관촉사 은진미륵(고려, 거대·파격·네모난 갓)** — 양식이 정반대.
3. **사진 단서 3종:** 벽돌 모양 돌 = 모전석탑 = **신라** / 8각·다층·대리석 10층 = **고려** / 이중기단 3층 정형 = **통일신라**.
