# 실전 워크북 ③ 고려 (건국 ~ 멸망)

> 한국사능력검정시험 **심화(2급 목표)** 대비 워크북.
> 대상 구간: 918 태조 왕건 즉위 ~ 신진사대부 성장(조선 건국 직전).
> 구성: ① 개념이해 → ② 실전문제 6문항 → ③ 정답 및 해설.

---

## 1. 개념이해

고려는 **정치 변동의 큰 흐름이 척추**다. 다음 한 줄 순서를 먼저 못 박는다.

> **문벌귀족 → 무신정권 → 대몽항쟁 → 원 간섭기 → 공민왕 개혁 → 신진사대부**

이 순서 위에 ① 초기 왕별 업적(태조·광종·성종), ② 대외관계(거란·여진·몽골), ③ 통치제도(도병마사·전시과)를 얹으면 고려 구간은 거의 끝난다. 시험은 단순 암기가 아니라 **"이 사건이 어느 단계인가", "이 키워드가 누구의 것인가"**를 한 칸 어긋나게 내는 식이므로, 흐름과 인물을 1:1로 묶는 훈련에 집중한다.

### (1) 초기 국가 기틀 — 왕별 업적 (가장 중요)

초기 세 왕은 **"통합 → 왕권 강화 → 유교 체제 완성"** 한 줄 인과로 읽으면 따로 노는 암기가 하나의 이야기가 된다.

| 왕 | 핵심 업적 (★=단골) | 방향 |
|---|---|---|
| **태조 왕건** | 호족 통합책 → **사심관 제도**(중앙 고관에게 출신지 관리 책임)·**기인 제도**(호족 자제를 볼모로 중앙에), ★**훈요 10조**(후대 왕 교훈), **북진 정책**(서경=평양 중시, 거란 배척 = **만부교 사건**), 흑창(빈민 구제) | 호족 통합 |
| **광종** | ★**노비안검법**(불법으로 노비 된 자 해방 → 호족 약화·왕권 강화), ★**과거제 실시**(쌍기 건의), **칭제건원**(황제 칭호, **광덕·준풍** 연호), 공복 제정 | 왕권 강화 |
| **성종** | ★**최승로 시무 28조** 수용 → 유교 통치 이념 확립, **2성 6부** 중앙 관제 정비, ★**12목 설치**(지방관 파견 시작), 국자감 정비 | 유교 체제 완성 |

> **흐름 한 줄:** 태조(통합) → 광종(왕권 강화) → 성종(유교 체제 완성).

### (2) 통치 체제 — 귀족 합의 정치

- **중앙**: 2성(중서문하성·상서성) 6부.
  - **도병마사**(국방·군사 회의) + **식목도감**(법·제도 제정) = 고려의 **독자적 합좌(재추) 회의 기구**(중서문하성+중추원 고관이 모여 결정). → **귀족 합의 정치의 상징**.
- **토지 — 전시과**: 관리에게 **전지(농경지) + 시지(임야)의 수조권** 지급. 정비 순서는 **경종 시정전시과 → 목종 개정전시과 → 문종 경정전시과**.
- **관리 등용**: 과거(문과 중시) + **음서**(공신·고위 관료 자손은 시험 없이 관직). → **문벌귀족 사회**의 특징.
- **군사**: 중앙 2군 6위, 지방 주현군·주진군.

> **혼동 금지:** **도병마사·식목도감 = 고려**의 독자 합좌 기구. 조선의 의정부·비변사와 섞지 말 것.

### (3) 정치 변동 (★ 시험의 척추 — 순서 암기 필수)

1. **문벌귀족기** — 음서·공음전으로 특권 세습.
   - **이자겸의 난(1126)**: 외척 이자겸이 왕권 위협 → 인종이 제거.
   - **묘청의 서경 천도 운동(1135)**: 묘청(서경파)이 ★**서경(평양) 천도 + 칭제건원 + 금국 정벌** 주장 → 개경파 **김부식**에게 진압. 국호 "대위", 연호 "천개". (신채호가 "조선 역사상 일천년래 제일대사건"으로 평가.)
2. **무신정권(1170~1270)** — **무신정변**(정중부 등, 차별 대우 반발)으로 시작. 권력기구 변천 → 최충헌이 ★**교정도감**(최고 권력기구) 설치, 최우가 ★**정방**(인사권 장악)·서방 설치, **삼별초**(군사 기반). 농민·천민 봉기(망이·망소이, 만적의 난).
3. **대몽 항쟁** — 몽골 침입 → 최우가 **강화도 천도**(장기 항전). ★**팔만대장경** 조판(부처 힘으로 외적 격퇴 기원). 김윤후(처인성·충주성). 개경 환도 반대 → ★**삼별초 항쟁**(강화 → 진도 용장성 → 제주 항파두리, 여·몽 연합군에 진압).
4. **원 간섭기** — 부마국으로 격하. 왕실 호칭 격하(조·종 → "충○왕"), 관제 격하(2성→첨의부). **정동행성**(내정 간섭 기구), 쌍성총관부·동녕부·탐라총관부(영토 상실). **권문세족** 등장(친원 세력, 대농장·음서). 변발·몽골풍 유행.
5. **공민왕 반원 자주 개혁(★)** — **쌍성총관부 무력 수복**(철령 이북 회복), **정동행성 이문소 폐지**, 친원파(기철 등) 숙청, 몽골풍 금지, 관제 복구. 내정 → ★**전민변정도감 설치**(신돈 등용, 권문세족이 빼앗은 토지·노비를 백성에게 환원).
6. **신진 사대부 성장** — 공민왕기 성균관 중심 + 과거로 진출, **성리학** 수용, 권문세족 비판 → 조선 건국 주도 세력으로 연결.

> **순서 못 박기:** 이자겸(1126) → 묘청(1135) → 무신정변(1170) → 강화 천도·대몽항쟁 → 원 간섭기 → 공민왕 → 신진사대부. **묘청이 무신정변보다 앞**임을 자주 헷갈리니 주의.

### (4) 대외 관계 — 인물·전투 매칭이 핵심

| 상대 | 인물 | 결과물 | 핵심 사건 |
|---|---|---|---|
| **거란(요)** | **서희** | **강동 6주**(압록강 방면) | 1차 침입 → **외교 담판**으로 획득 |
| **거란(요)** | **강감찬** | 이후 나성·천리장성 축조 | 3차 침입 → ★**귀주 대첩(1019)** 대승 |
| **여진(금)** | **윤관** | **동북 9성**(후에 반환) | ★**별무반**(신기군·신보군·항마군) 편성 → 정벌 |
| **몽골** | 최우·김윤후·삼별초 | 강화 천도·팔만대장경 | (위 (3)-3 대몽 항쟁 참조) |

> 거란 2차 침입은 현종 때 개경이 함락되었다. 이후 금이 군신 관계를 요구하자 **이자겸이 수용**했다(여진 → 금으로 성장).

### (5) 문화 — 단골 문화재

- **인쇄·기록**
  - **팔만대장경(재조대장경)**: **몽골** 격퇴 기원 → 합천 해인사 장경판전에 **현존**.
  - **초조대장경**: **거란** 격퇴 기원 → 몽골 침입 때 **소실**.
  - **직지심체요절(직지)**: 현존 **세계 最古 금속활자본**, 청주 흥덕사, 현재 **프랑스 국립도서관** 소장.
- **불교**
  - **의천 — 천태종**: 교종 중심으로 교선 통합, **교관겸수**.
  - **지눌 — 조계종/수선사 결사**: 선종 중심, **정혜쌍수·돈오점수**. (혜심은 유불일치설.)
- **공예**: **고려청자** — 비취색 순청자 → **상감청자**로 발전(상감 기법은 고려 독자 기술).
- **역사서**
  - 김부식 **삼국사기**: 현존 最古 역사서, **기전체**, 유교적·신라 계승 의식.
  - 일연 **삼국유사**: 불교·설화 중심, **단군 신화 최초 수록**.
  - 이규보 **동명왕편**(고구려 계승), 이승휴 **제왕운기**.
- **건축**: 주심포 양식(영주 부석사 무량수전·안동 봉정사 극락전), 월정사 8각 9층 석탑, 논산 관촉사 석조미륵보살입상(대형 불상).

> **3대 혼동 함정(반드시 구분)**
> - **팔만대장경(몽골·현존) ↔ 초조대장경(거란·소실)**: "침입한 적 + 보존 여부"를 세트로.
> - **의천(천태종·교관겸수) ↔ 지눌(조계종·정혜쌍수·돈오점수)**: "교"가 들어가면 의천(교종 바탕).
> - **강동 6주(거란·서희·외교) ↔ 동북 9성(여진·윤관·별무반)**: "6주/9성", "거란/여진", "서희/윤관" 교차 오답.

## 3. 정답 및 해설

### 문제 1 — 정답 ② 광종

- **자료 식별:** 과거제(쌍기 건의) + 노비안검법 + 칭제건원(광덕·준풍 연호) → 셋 다 **광종**의 왕권 강화 정책.
- **정답 근거:** 노비안검법으로 호족의 경제·군사 기반을 약화시키고, 과거제로 새 인재를 등용했으며, 황제를 칭하고 독자 연호를 써 왕권을 과시했다.
- **오답 반박**
  - ① 태조 — 사심관·기인·훈요 10조·북진 정책이 핵심. 노비안검법·과거제와 무관.
  - ③ 성종 — 시무 28조·12목·2성 6부 정비가 핵심.
  - ⑤ 공민왕 — 쌍성총관부 수복·전민변정도감(신돈) 등 반원 개혁. 연호(광덕·준풍)는 광종.
- **한 줄 키워드:** 노비안검법 + 과거제 + 광덕·준풍 = **광종**.

### 문제 2 — 정답 ③ 2성 6부의 중앙 관제를 정비하였다

- **자료 식별:** 시무 28조(최승로) + 12목 설치 → **성종**.
- **정답 근거:** 성종은 유교 통치 이념을 확립하면서 **2성 6부**의 중앙 관제를 정비하고 국자감을 정비했다. 따라서 ③이 성종의 업적으로 옳다.
- **오답 반박**
  - ① 사심관·기인 제도 → **태조**.
  - ② 노비안검법 → **광종**.
  - ④ 쌍성총관부 수복 → **공민왕**.
  - ⑤ 전민변정도감·신돈 등용 → **공민왕**.
- **한 줄 키워드:** 시무 28조 + 12목 = **성종** → 2성 6부 정비.

### 문제 3 — 정답 ① (나) - (라) - (가) - (다)

- **정답 근거:** 정치 변동의 순서는 **이자겸의 난(1126) → 묘청 서경 천도 운동(1135) → 무신정변(1170) → 강화 천도(대몽 항쟁, 1232)**.
  - (나) 이자겸(1126) → (라) 묘청(1135) → (가) 무신정변(1170) → (다) 강화 천도.
- **오답 반박**
  - ② 묘청(라)과 무신정변(가)의 순서를 뒤바꿈 — 묘청(1135)이 무신정변(1170)보다 **앞**이다. (가장 잘 나오는 함정.)
  - ③·⑤ 묘청(라)을 이자겸(나)보다 앞세운 오류 — 이자겸(1126)이 먼저다.
  - ④ 강화 천도(다)와 무신정변(가)의 순서를 뒤바꿈 — 무신정변(1170)이 강화 천도(1232)보다 앞.
- **한 줄 키워드:** 이자겸 → 묘청 → 무신정변 → 강화 천도. **묘청이 무신정변보다 앞**.

### 문제 4 — 정답 ③ 외교 담판으로 강동 6주를 획득하였다

- **자료 식별:** 거란 소손녕 + 담판 + 강동 6주 → **서희**.
- **정답 근거:** 서희는 거란 1차 침입 때 소손녕과의 **외교 담판**으로 거란군을 물러가게 하고 **강동 6주**를 얻어 압록강 방면으로 영토를 넓혔다. 자료 내용과 그대로 일치하므로 ③.
- **오답 반박**
  - ① 별무반·여진 정벌 → **윤관**.
  - ② 귀주 대첩(거란 3차 침입) → **강감찬**. (같은 거란 상대라 가장 헷갈리는 함정 — 서희=외교/강동 6주, 강감찬=전투/귀주.)
  - ④ 동북 9성 축조·반환 → **윤관**(여진).
  - ⑤ 처인성에서 몽골 장수(살리타) 사살 → **김윤후**.
- **한 줄 키워드:** 거란 + 담판 + 강동 6주 = **서희**.

### 문제 5 — 정답 ② 쌍성총관부를 공격하여 철령 이북의 땅을 되찾았다

- **자료 식별:** 기철 일파 숙청 + 몽골풍 금지 + 관제 복구 → 반원 자주 개혁 → **공민왕**.
- **정답 근거:** 공민왕은 **쌍성총관부를 무력으로 수복**하여 철령 이북의 땅을 되찾았다. 자료의 반원 개혁과 같은 맥락이므로 ②가 옳다.
- **오답 반박**
  - ① 노비안검법, ⑤ 과거제 시행 → 모두 **광종**.
  - ③ 시무 28조 수용 → **성종**.
  - ④ 강화도 천도·대몽 항전 → **최우(무신정권)**. 같은 '몽골(원)' 관련이라 헷갈리지만, 강화 천도는 무신정권기의 대몽 항쟁이고 공민왕은 그 후 원 간섭기를 끝내는 개혁임에 주의.
- **한 줄 키워드:** 기철 숙청 + 몽골풍 금지 = **공민왕** → 쌍성총관부 수복.

### 문제 6 — 정답 ⑤ 일연의 삼국사기는 단군 신화를 최초로 수록한 역사서이다

- **자료 식별:** "옳지 **않은** 것"을 고르는 문제임에 주의.
- **정답 근거:** **단군 신화를 최초로 수록한 책은 일연의 『삼국유사』**다. 『삼국사기』는 **김부식**이 지은 유교적 기전체 역사서로, 단군 신화를 싣지 않았다. 책 이름과 저자·내용이 어긋나므로 ⑤가 틀린 설명 → 정답.
- **오답(= 모두 옳은 설명) 확인**
  - ① 팔만대장경(재조대장경) = 몽골 격퇴 기원 — 맞음. (초조대장경은 거란 격퇴 기원 → 몽골 침입 때 소실.)
  - ② 직지심체요절 = 현존 세계 最古 금속활자본(프랑스 국립도서관 소장) — 맞음.
  - ③ 의천 = 천태종·교관겸수 — 맞음.
  - ④ 지눌 = 조계종·정혜쌍수·돈오점수 — 맞음.
- **한 줄 키워드:** 단군 신화 최초 수록 = **삼국유사(일연)**. 삼국사기(김부식)는 기전체. (의천=천태종·교관겸수 ↔ 지눌=조계종·정혜쌍수·돈오점수 함정도 한 번 더 점검.)

### ⚠️ 시험장 직전 4대 함정 체크

1. **왕별 키워드 1:1** — 노비안검법·과거제·광덕/준풍 = **광종** / 시무 28조·12목 = **성종** / 사심관·기인·훈요 10조 = **태조**.
2. **정치 변동 순서** — 이자겸(1126) → **묘청(1135)** → 무신정변(1170) → 강화 천도(1232). **묘청이 무신정변보다 앞**.
3. **대외관계 교차 오답** — 거란·서희·강동 6주·외교 담판 ↔ 여진·윤관·동북 9성·별무반 / 거란 3차는 강감찬·귀주 대첩.
4. **문화 3대 함정** — 팔만대장경(몽골·현존) ↔ 초조대장경(거란·소실), 의천(천태종·교관겸수) ↔ 지눌(조계종·정혜쌍수·돈오점수), 단군 신화 = 삼국유사(일연).
