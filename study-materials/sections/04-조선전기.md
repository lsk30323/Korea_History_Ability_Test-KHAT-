# 시대④ 조선 전기 (건국 ~ 임진왜란 이전)

> 한능검 **심화(2급 목표)** 대비 정리. 통치체제·사림·세종 문화가 핵심.
> 대상 구간: 1392 태조 즉위 ~ 1592 임진왜란 직전.

---

## ① 기출 비중 (최근 회차 기준)

- **평균 문항 수: 6~8문항(추정).** 조선 전기는 매 회차 꾸준히 출제되는 안정적 고득점 구간이다.
  - 정확한 회차별 시대 구분 통계는 공식(historyexam.go.kr)에서 문항 단위로 공개하지 않아 **수치는 추정**이며, 기출 해설·분석 자료(공기출·EBS 등)와 일반적 출제 경향을 종합한 값이다.
- **전근대사 : 근현대사 ≈ 60 : 40** (검색 교차 확인됨). 조선(전+후기)이 전근대 안에서 가장 비중이 크다.
- **반복 출제율이 매우 높다(체감 90~95%).** 보기 지문이 회차를 넘어 거의 그대로 재출제되므로, 단골 키워드만 잡아도 합격선(80%)에 충분히 닿는다.

### 단골 출제 소재 빈도 (체감, 추정)

| 소재 | 빈도 | 자주 나오는 형태 |
|---|---|---|
| **세종의 업적/문화** | 매우 높음 | 인물·정책 "옳은 것" 고르기 (훈민정음·집현전·칠정산·측우기·4군6진) |
| **4대 사화 + 조광조(기묘)** | 매우 높음 | 사건 순서 배열, "(가) 사화" 식별 |
| **경국대전·통치기구(3사·의정부6조)** | 높음 | 기구 기능 매칭, 성종 업적 |
| **직전법 / 관수관급제 등 토지제도** | 높음 | 수취·토지제도 변천 순서 |
| **사림 vs 훈구 / 서원·향약** | 중상 | 정치 세력 비교, 향촌 자치 |
| **대외관계(사대교린·3포·4군6진)** | 중 | 명·여진·일본 정책 구분 |
| **편찬·과학(농사직설·동국통감·동국여지승람·국조오례의·악학궤범)** | 중 | 서적-왕대 매칭 |

> 즉, **"세종 + 4대 사화 + 경국대전·3사 + 토지제도"** 4개 묶음이 조선 전기 점수의 대부분을 만든다.

## ② 2급 핵심 요약 (흐름 중심)

### (1) 왕별 업적 — 가장 자주 나오는 뼈대

| 왕 | 핵심 키워드 (★=단골) |
|---|---|
| **태조** | 조선 건국(1392), 한양 천도, **정도전**(재상중심 정치, 『조선경국전』, 불교 비판 『불씨잡변』) |
| **태종** | **6조 직계제**(왕권 강화), **호패법**, 사병 혁파, 신문고, 양전·사원전 정리 |
| **세종** | ★**집현전**, ★**훈민정음** 창제, **의정부 서사제**(재상에게 권한 위임), **칠정산**(역법), 측우기·앙부일구·자격루, **『농사직설』**, **4군 6진**(최윤덕·김종서, 압록강~두만강), 쓰시마 정벌(이종무), 3포 개항 |
| **세조(수양대군)** | 계유정난으로 집권, ★**6조 직계제 부활**, **직전법**(현직 관리에게만 수조권), 집현전·경연 폐지, 『경국대전』 편찬 **시작** |
| **성종** | ★**『경국대전』 완성·반포**(통치 규범 확립), ★**홍문관** 설치(집현전 계승, 경연 주관), 『동국통감』·『동국여지승람』·『국조오례의』·『악학궤범』 편찬 |

> 함정 주의: **경국대전은 세조 때 "편찬 시작" → 성종 때 "완성·반포".** 시험은 "완성=성종"을 묻는다.

### (2) 통치 체제

- **중앙**: **의정부**(재상 합의·국정 총괄) + **6조**(이·호·예·병·형·공, 실무 집행).
- **3사(언론 기구)**: **사헌부**(감찰), **사간원**(간쟁·봉박), **홍문관**(경연·학문·정책 자문). → 권력 독점 견제. 사헌부+사간원=**양사(대간)**, 서경권 행사.
- **지방**: 전국 **8도** → 부·목·군·현. **모든 군현에 수령(지방관) 파견**(고려와의 핵심 차이), 향리는 수령 보좌(지위 격하). 유향소(향촌 자치)·경재소.
- **군사**: 5위(중앙), 진관 체제(지방).
- **과거**: 문과(대과)·무과·잡과. 문과 중시. **승과는 점차 폐지.** + 음서·천거 보조.

### (3) 사림의 성장과 4대 사화 — 순서가 생명

- **훈구**: 세조 즉위 공신 계열, 기득권. **사림**: 지방 중소지주·성리학 명분 중시, 성종 때 김종직 등용으로 중앙 진출(주로 3사 언관직).
- **4대 사화 (순서 암기 필수)**

| 사화 | 왕 | 핵심 원인/사건 |
|---|---|---|
| **무오사화**(1498) | 연산군 | **김종직의 「조의제문」**(사초 문제), 훈구가 사림 공격 |
| **갑자사화**(1504) | 연산군 | **폐비 윤씨 사사 사건** 보복 |
| **기묘사화**(1519) | 중종 | ★**조광조**의 개혁(현량과·위훈 삭제·소격서 폐지) 반발 → "주초위왕(走肖爲王)" 모함 |
| **을사사화**(1545) | 명종 | 외척 **대윤(윤임) vs 소윤(윤원형)** 권력 다툼 |

> 암기 두문자 추천: **무–갑–기–을 / 연–연–중–명** (연산군 두 번 → 중종 → 명종).

- 사화로 큰 타격을 입은 사림은 **향촌으로 낙향 → 서원·향약을 기반으로 세력 재건 → 선조 때 정계 장악(붕당 정치로 연결)**.
  - **서원**: 선현 제사 + 후학 교육 (최초 **백운동 서원**, 주세붕 → 이황 건의로 **소수서원** 사액).
  - **향약**: 향촌 자치 규약(덕업상권·과실상규·예속상교·환난상휼), 풍속 교화·사림의 농민 통제 기반.

### (4) 대외 관계 — 사대교린

- **사대(명)**: 조공·책봉 형식의 친선·실리 외교(자주성 유지).
- **교린(여진·일본)**: 회유(무역소·3포)와 강경(정벌) 병행.
  - 여진: **4군 6진** 개척(세종, 영토를 압록강~두만강까지 확장), 사민·토관 정책.
  - 일본: 세종 때 **쓰시마(대마도) 정벌**(이종무) → **3포 개항**(부산포·제포·염포) → 계해약조로 제한 무역. 이후 **삼포왜란·을묘왜변** → **비변사** 설치 계기.

### (5) 문화·사상

- **성리학** 국가 통치 이념화. **이황(『성학십도』, 주리론, 영남학파)** · **이이(『성학집요』, 주기론, 기호학파)** — 단, 본격 학문 논쟁은 16세기.
- 과학·편찬: 세종대 **칠정산·『농사직설』·측우기·자격루**, 성종대 **『동국통감』(역사)·『동국여지승람』(지리)·『국조오례의』(의례)·『악학궤범』(음악)**, 태조대 **혼일강리역대국도지도**(현존 동양 최고 세계지도).
- 공예: **분청사기**(15세기) **→ 순백자(16세기)**로 변화. (조선 전기는 백자/분청, 청화백자는 후기 유행.)
- 의궤·인쇄: 활자(계미자·갑인자) 개량.

## ③ 암기 카드 / 문제 (Q/A, 12개)

**Q1.** 태종이 왕권 강화를 위해 시행한, 6조가 의정부를 거치지 않고 왕에게 직접 보고하던 제도는?
**A1.** **6조 직계제**. (세조 때 부활, 세종은 의정부 서사제로 전환)

**Q2.** 세종이 학문 연구·정책 자문을 위해 설치·육성한 기구는? 또 그 학문 기능을 계승해 성종이 세운 기구는?
**A2.** **집현전** / 계승 기구는 **홍문관**.

**Q3.** 세종 때 김종서·최윤덕이 여진을 몰아내고 개척하여 영토를 압록강~두만강까지 넓힌 것은?
**A3.** **4군 6진**.

**Q4.** 조선의 기본 법전으로, 세조 때 편찬을 시작해 **성종 때 완성·반포**된 통치 규범은?
**A4.** **『경국대전』**.

**Q5.** 세조가 토지 부족을 해결하기 위해 **현직 관리에게만** 수조권을 지급한 토지제도는?
**A5.** **직전법**. (이후 명종 때 직전법 폐지 → 녹봉제로 전환)

**Q6.** 조선의 3사를 모두 쓰고 각 기능을 한 단어로 답하라.
**A6.** **사헌부**(감찰)·**사간원**(간쟁)·**홍문관**(경연·자문).

**Q7.** 4대 사화를 일어난 순서대로, 각 왕과 함께 나열하라.
**A7.** **무오(연산군) → 갑자(연산군) → 기묘(중종) → 을사(명종)**.

**Q8.** 중종 때 현량과 실시·위훈 삭제 등 급진 개혁을 펴다 기묘사화로 제거된 사림의 대표 인물은?
**A8.** **조광조**. (모함 문구: "주초위왕")

**Q9.** 무오사화의 직접 발단이 된, 김종직이 쓴 항우의 의제 살해를 빗댄 글은?
**A9.** **「조의제문」**.

**Q10.** 향촌 자치 규약으로, 덕업상권·과실상규·예속상교·환난상휼을 4대 덕목으로 하는 것은?
**A10.** **향약**.

**Q11.** 세종 때 시행한, 우리 실정에 맞게 한양 기준으로 만든 역법서는?
**A11.** **칠정산**(내편·외편).

**Q12. [함정 주의]** 일본과 관련해 세종 때의 '3포 개항'과, 조선 후기 임진왜란을 거치며 국정 최고 기구가 된 기구를 구분하라.
**A12.** **3포 개항**=부산포·제포·염포(세종, 제한적 무역 허용). **국정 최고 기구**=**비변사**(여진·왜구 대비 임시기구로 출발 → 임진왜란 후 상설·최고 기구화). 둘을 섞지 말 것.

### 자주 틀리는 함정 정리 (꼭 확인)

1. **경국대전 = 성종 "완성".** 세조는 '시작'일 뿐. ("세조가 완성"으로 낚는 보기 주의.)
2. **집현전 → 홍문관 → (임진왜란 후) 비변사** 계보 혼동 금지. 집현전·홍문관은 학문/언론, 비변사는 군사·국정 총괄(후기 핵심).
3. **6조 직계제(태종·세조, 왕권↑) vs 의정부 서사제(세종, 재상권↑)** — 시행 왕과 권력 방향을 반대로 적기 쉬움.
4. **분청사기·백자(전기) vs 청화백자(후기)** — 도자기 시기 구분.
