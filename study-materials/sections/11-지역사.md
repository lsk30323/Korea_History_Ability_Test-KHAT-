# 주제 ⑪ 지역사 (地域史) — 한능검 2급 대비

> "다음 (가) 지역에 대한 설명으로 옳은 것은?" / "(가) 지역에서 있었던 사실로 옳은 것은?"
> 한 **지역을 고정**해 두고 **여러 시대의 사건을 묶어** 매칭시키는 단골 유형. 시대 구분 능력 + 지역 키워드 암기가 핵심.
> 본 문서는 지역별로 사건을 **시대순 타임라인**으로 정리한다.

---

## ① 기출 비중 (출제 빈도)

> ⚠️ **수치는 추정치**입니다. 한국사능력검정시험 공식 사이트(historyexam.go.kr)와 기출 분석 자료를 교차 확인했으나, 회차별 세부 문항 분류표 원본(PDF) 일부는 접근이 차단되어 정확한 카운트 대신 경향 기반 추정으로 제시합니다.

| 구분 | 추정 출제량 (심화 50문항 기준) | 비고 |
|---|---|---|
| **지역 묶음형 문항** ("(가) 지역…") | 회당 **1문항 (약 2%)** | 거의 매회 1문항 고정 출제 |
| **독도 단독 문항** | **거의 매회 1문항 (2%)** | "독도가 우리 영토인 근거" 단골. 사실상 고정 |
| 특정 지역 연계 사건 (지역명이 보기·발문에 등장) | 회당 2~4문항에 **간접 등장** | 강화도·평양·개성 등은 다른 주제(대몽항쟁·개항·근현대) 안에서 자주 등장 |

**단골 출제 지역 빈도 (추정, 높은 순):**

1. **강화도** — 압도적 1순위. 선사~근대까지 사건 밀도가 최고라 "한 지역 = 한국사 전체"를 묻기 좋아 자주 출제.
2. **독도** — 별도 단독 문항으로 거의 매회 출제 (영토주권 관련 정책 의지).
3. **평양 / 서경** — 고조선·고구려·묘청·근현대 묶기 좋음.
4. **개성** — 고려 수도 + 근현대(개성공단) 연계.
5. **제주** — 삼별초 + 4·3 + 하멜 묶기.
6. **부산(동래)** — 임진왜란 + 개항(1876) + 6·25 임시수도.
7. **간도·연해주** — 독립운동 파트에서 자주.
8. **거문도** — "영국 점령" 단일 키워드로 가끔.

> **학습 우선순위:** 강화도 ≫ 독도 > 평양 ≈ 개성 ≈ 제주 > 부산 > 간도·연해주 > 거문도.
> 강화도·독도 두 곳만 완벽히 정리해도 지역사 문항 대부분을 잡는다.

## ② 2급 핵심 요약 (지역별 시대순 타임라인)

각 지역은 **"이 지역 = 이 키워드"** 로 묶어서 외운다. 연도보다 **사건의 시대 순서**가 중요.

### 🏝️ 강화도 — 선사부터 근대까지 한국사의 축소판 (★최우선)

| 시대 | 사건 / 유적 | 핵심 키워드 |
|---|---|---|
| 선사(청동기) | **고인돌** (부근리 탁자식) | 유네스코 세계유산, 지배층 무덤 |
| 고조선·단군신화 | **마니산 참성단** | 단군이 하늘에 제사 지냈다는 제천단 |
| 고려 (1232~1270) | **강화 천도** — 최우의 대몽항쟁 | 39년간 임시수도, **팔만대장경** 조판 시작 |
| 조선 후기 (1866) | **병인양요** | 프랑스군 침입 → **정족산성**에서 **양헌수** 격퇴, 외규장각 도서 약탈 |
| 조선 후기 (1871) | **신미양요** | 미군 침입 → **광성보**에서 **어재연** 전사 (수자기 약탈) |
| 조선 후기 (1876) | **강화도 조약(조일수호조규)** | 최초 근대적·불평등 조약, **연무당**에서 체결, 운요호 사건이 빌미 |

> 강화도는 **시대순 매칭**이 정답 포인트: 청동기(고인돌) → 고려(천도·대장경) → 1866 병인양요 → 1871 신미양요 → 1876 강화도조약.

### 🪨 독도 — 영토주권의 역사적 근거 (★거의 매회 단독 출제)

| 시대 | 사건 / 인물 | 핵심 |
|---|---|---|
| 신라 지증왕 (512) | **이사부**가 **우산국** 복속 | 울릉도·독도가 신라 영토로 편입된 시초 |
| 조선 숙종 (1693~) | **안용복** | 일본에 건너가 울릉도·독도가 조선 땅임을 확인받음 |
| 대한제국 (1900) | **칙령 제41호** | 울릉도를 **울도군**으로 승격, 죽도·**석도(石島=독도)**를 관할로 명시 |
| 일제 (1905) | 일본이 **시마네현 고시 제40호**로 불법 편입 | 러일전쟁 중 강제 편입 → **국제법상 무효** |

> **일본 편입 논리 반박:** "주인 없는 땅(무주지)을 선점했다"는 일본 주장 → 칙령 제41호(1900)가 1905년 일본 고시보다 **5년 앞서** 독도를 우리 행정구역으로 명시했으므로 무주지가 아니었음.

### 🏛️ 서울(한양) — 조선의 도읍, 근현대 정치의 무대

- 조선 태조 이성계가 **한양 천도(1394)** → 경복궁(법궁) 등 **5대 궁궐**(경복·창덕·창경·경희·덕수).
- 근현대: 운현궁(흥선대원군), 독립협회 **만민공동회**, 3·1 운동(탑골공원), 4·19 혁명, 6월 민주항쟁 등.
- 궁궐 매칭: **경복궁**(조선 법궁, 일제 조선총독부 설치) / **덕수궁**(아관파천 후 환궁, 대한제국 선포 무대, 석조전) / **창덕궁**(유네스코 세계유산).

### 🏯 개성 — 고려의 수도

- **고려 수도(개경/송악)**, 궁궐터 **만월대**, 태조 왕건릉.
- 공민왕릉, **선죽교**(정몽주 피살지).
- 근현대: **개성공단**(남북 경제협력 상징).

### ⛰️ 평양(서경) — 고조선·고구려·고려 서경

| 시대 | 사건 | 핵심 |
|---|---|---|
| 고조선 | **왕검성** (고조선 중심지로 추정) | |
| 고구려 (427) | **장수왕 평양 천도** | 남진 정책 |
| 고려 (1135) | **묘청의 서경 천도 운동** | 풍수지리, 칭제건원·금국정벌 주장 → 김부식 관군이 진압 |
| 근현대 | 제너럴셔먼호 사건(1866), 물산장려운동 시작(1920, 조만식) | |

### ⚓ 부산(동래) — 임진왜란·개항·전쟁

- **임진왜란(1592):** **동래성 전투**(송상현 순절), 부산진 전투(정발).
- 조선 후기: **초량 왜관**(대일 무역 창구).
- **개항(1876):** 강화도 조약으로 **최초 개항지**.
- 6·25: **임시수도**(1950~1953), 유엔기념공원.

### 🍊 제주 — 삼별초·하멜·4·3

| 시대 | 사건 | 핵심 |
|---|---|---|
| 고려 (1271~1273) | **삼별초** 최후 항쟁 (**항파두리성**, 김통정) | 진도 → 제주로 이동, 여몽연합군에 진압 |
| 원 간섭기 | **탐라총관부** 설치, 목마장 | |
| 조선 (1653) | 네덜란드인 **하멜** 표류 → 『하멜 표류기』 | |
| 현대 (1948) | **제주 4·3 사건** | 단독선거·단독정부 반대, 무력 충돌·진압 과정에서 다수 주민 희생 |

### 🌲 간도·연해주 — 독립운동 기지

- **간도(북간도):** 서전서숙·명동학교(민족교육), **봉오동·청산리 대첩**(1920), 간도참변(경신참변, 1920).
- **연해주(블라디보스토크):** **신한촌**, **대한광복군 정부**(1914, 이상설·이동휘), 1937년 **중앙아시아 강제 이주**(스탈린).
- **백두산정계비(1712):** 숙종 때 청과 국경 획정 → "토문강" 해석 논쟁(간도 귀속 문제).

### 🇬🇧 거문도 — 영국 점령

- **거문도 사건(1885~1887):** 영국이 **러시아 남하 견제**를 명분으로 거문도를 불법 점령. 조선은 협상에서 배제 → 조선의 취약한 국제적 지위를 보여줌.

## ③ 암기 카드 / 문제 (Q&A) — 최소 10개

#### Q1. (가) 지역: 고인돌이 있고, 고려가 대몽항쟁기에 도읍을 옮겼으며, 병인양요·신미양요가 일어난 곳은?
**A.** **강화도.** (선사 고인돌 → 고려 강화 천도 → 1866 병인양요 → 1871 신미양요 → 1876 강화도조약까지 한국사 전 시대가 압축된 지역.)

#### Q2. 신라 지증왕 때 우산국(울릉도·독도)을 복속시킨 인물은?
**A.** **이사부.** (512년. 이후 안용복(숙종) → 대한제국 칙령 제41호(1900)로 독도 영유 근거가 이어진다.)

#### Q3. 1900년 울릉도를 울도군으로 승격시키고 독도(석도)를 관할로 명시한 법령은?
**A.** **대한제국 칙령 제41호.** (일본의 시마네현 고시(1905)보다 5년 앞서 → "무주지 선점론" 반박의 핵심 근거.)

#### Q4. 고려 인종 때 풍수지리설을 내세워 평양(서경)으로 수도를 옮기자고 주장하며 칭제건원·금국정벌을 외친 운동은?
**A.** **묘청의 서경 천도 운동(1135).** (김부식의 관군에게 진압됨. 신채호가 "조선 역사상 일천년래 제일대사건"으로 평가.)

#### Q5. 고려의 수도였고 궁궐터인 만월대, 정몽주가 피살된 선죽교가 있는 지역은?
**A.** **개성(개경).** (근현대에는 남북 경제협력의 상징인 개성공단이 위치.)

#### Q6. 삼별초가 진도에 이어 최후로 항쟁한 곳(항파두리성)이자, 1948년 단독선거에 반대해 일어난 사건의 무대인 지역은?
**A.** **제주도.** (삼별초 대몽항쟁(1273) + 제주 4·3 사건(1948). 그 사이 1653년 하멜 표류도 제주.)

#### Q7. 임진왜란 때 송상현이 순절한 동래성 전투가 벌어졌고, 1876년 최초로 개항된 지역은?
**A.** **부산.** (임진왜란 → 초량 왜관 → 1876 최초 개항 → 6·25 임시수도로 이어지는 타임라인.)

#### Q8. 1885년 영국이 러시아의 남하를 견제한다는 구실로 불법 점령한 섬은?
**A.** **거문도.** (거문도 사건, 1885~1887. 조선은 외교 협상에서 철저히 배제됨.)

#### Q9. 1920년 봉오동·청산리 대첩이 벌어졌고, 명동학교 등 민족교육이 이루어진 독립운동 기지는?
**A.** **간도(북간도).** (연해주 신한촌과 함께 무장 독립운동의 양대 거점.)

#### Q10. 병인양요 때 프랑스군을 정족산성에서 격퇴한 인물과, 신미양요 때 광성보에서 전사한 인물은 각각 누구인가?
**A.** 병인양요 = **양헌수**(정족산성) / 신미양요 = **어재연**(광성보). (둘 다 강화도. 어재연의 '수자기'를 미군이 약탈해 갔다.)

#### Q11. 조선 태조가 천도한 도읍이며, 경복궁·창덕궁 등 5대 궁궐이 있는 곳은?
**A.** **서울(한양).** (덕수궁은 아관파천 후 고종이 환궁해 대한제국을 선포한 무대.)

### ⚠️ 자주 틀리는 함정 카드

#### 함정 Q1. **병인양요(1866) vs 신미양요(1871)** — 침입국과 격전지를 바꿔 출제!
**A.** **병인**양요 = **프랑스**, 양헌수, **정족산성**, 외규장각 도서 약탈. **신미**양요 = **미국**, 어재연, **광성보**, 수자기 약탈.
👉 "병인=프랑스=정족산성 / 신미=미국=광성보"로 묶어 암기. (둘 다 장소는 강화도라 헷갈림.)

#### 함정 Q2. **강화도 조약(1876) ≠ 강화도에서 일어난 다른 사건**
**A.** 강화도는 사건이 너무 많아, "강화도조약 체결지"를 묻는데 병인·신미양요 보기로 헷갈리게 함. **강화도조약 = 1876년, 연무당, 운요호 사건이 빌미, 최초의 근대적 불평등 조약, 부산·원산·인천 개항.** 양요(1866·1871)는 그보다 **앞선** 사건임을 시대순으로 구분.

#### 함정 Q3. **서경(西京)은 평양** / 동경=경주 / 남경=한양(서울)
**A.** 고려의 3경에서 **서경 = 평양**. 묘청이 천도하려던 곳은 '서경=평양'이지 개성(개경)이 아니다. (개경은 묘청이 **벗어나려** 한 보수 문벌의 본거지.)
