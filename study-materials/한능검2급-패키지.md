# 한국사능력검정시험 2급 합격 통합 패키지

> **목표: 심화 50문항 중 안정권 40문항(80%) 확보.** 핵심 영역(고대·고려·조선·근대)을 최우선으로 회독한다.
> 전근대 : 근현대 ≈ 60 : 40 구성. 큰 흐름 + 단골 유형 + 함정 카드에 화력을 집중하고, 1급용 지엽(세부 연대·가마터·세부 토기명 등)은 과감히 생략한다.
> 13개 검수 항목 전원 합격(13/13, 핵심 5영역 전부 통과)을 확인하고 조립한 최종본. 출제 빈도 수치는 공식 통계 부재로 **최근 회차(60~77회) 경향 기반 추정치**임을 밝혀 둔다.

# 2부. 시대별 핵심 요약 (01~08 흐름 순서 병합)

## ① 선사 ~ 국가의 형성

**선사 4단계 (도구·토기·생활):**
- **구석기**: 뗀석기(주먹도끼·찍개), 동굴·막집, 이동 생활.
- **신석기**: 간석기·가락바퀴·뼈바늘, **빗살무늬토기**, **농경·목축 시작**, 정착(움집), 평등사회.
- **청동기**: 반달돌칼·비파형동검, 미송리식 토기, **계급·군장 발생**, **고인돌**.
- **철기**: 철제 농기구, **거푸집**, 세형동검, 명도전(중국 교류), 독무덤.
- 필수 구분: 농경 시작 = 신석기 / 계급 발생 = 청동기 / 비파형동검 = 청동기·세형동검 = 철기.

**고조선:** 단군왕검(제정일치) → 위만조선(철기 본격 수용·중계무역). **8조법**(살인=사형, 상해=곡식 배상, 절도=노비). 한 무제 침공으로 멸망 → 한 군현(낙랑·진번·임둔·현도).

**여러 나라(풍습 매칭):** 부여(사출도·영고·1책 12법) / 고구려(제가회의·동맹·서옥제) / 옥저(민며느리제·가족공동무덤, 제천행사 없음) / 동예(무천·책화·단궁·과하마·반어피) / 삼한(천군·소도=제정 분리, 변한=철). 제천행사 두문자 "영동무"(부여·고구려·동예).

## ② 고대 (삼국·가야·통일신라·발해) — 최다 출제 ★★★

**삼국 전성기 (4→5→6세기 = 백→고→신):**
- 백제 **근초고왕**(4C): 마한 정복, 고구려 평양성 공격(고국원왕 전사), 칠지도. → **무령왕**(22담로) → **성왕**(사비 천도·남부여, 관산성 전사).
- 고구려 **광개토대왕**(영락, 신라 침입 왜 격퇴 400) → **장수왕**(평양 천도 427·남진, 한성 함락, 광개토대왕릉비·충주고구려비).
- 신라 **지증왕**(국호 신라·우산국 복속) → **법흥왕**(율령·불교 공인·병부·금관가야 병합, 건원) → **진흥왕**(한강 전역 장악·대가야 정복·화랑도·순수비).

**가야:** 전기 금관가야(김해, 철·중계무역, 법흥왕에 멸망 532) → 후기 대가야(고령, 진흥왕에 멸망 562). 연맹 단계에서 멸망.

**통일 과정(순서 필수):** 살수대첩(612) → 안시성(645) → 나당동맹(648) → 백제 멸망(660) → 고구려 멸망(668) → 매소성(675)·기벌포(676) → 삼국통일(676). 부흥운동은 멸망 직후 발생.

**통일신라:** 무열왕(최초 진골) → 문무왕(통일 완성) → **신문왕**(김흠돌 난 진압·9주5소경·9서당10정·**관료전 지급(687)·녹읍 폐지(689)**·국학). 토지 함정: 관료전/녹읍 폐지=신문왕, 정전=성덕왕, **녹읍 부활=경덕왕**. 신라 말: 호족·6두품·선종·풍수지리·원종애노의 난 → 후삼국.

**발해(남북국):** 대조영 건국(698, 고구려 유민+말갈) → 무왕(장문휴 산둥 공격, 인안) → 문왕(상경 천도, 대흥) → 선왕(5경 15부 62주, **해동성국**). 고구려 계승 근거: 일본 국서 '고려국왕' 자칭, 온돌·굴식돌방무덤·이불병좌상.

**고대 문화:** 무구정광대다라니경(석가탑, 최고 목판 인쇄물), 백제 금동대향로, 일본 전파(노리사치계=불교/왕인=천자문/담징=종이·먹·호류사 벽화).

## ③ 고려 ★★★

**왕별 업적(초기 기틀):** **태조**(사심관·기인·훈요10조·북진) → **광종**(노비안검법·과거제·칭제건원) → **성종**(최승로 시무 28조·2성6부·12목).

**통치 체제:** 2성 6부 + **도병마사·식목도감**(독자적 재추 합좌). **전시과**(전지+시지 수조권). **음서**(문벌귀족 특징).

**정치 변동(척추, 순서 필수):** 문벌귀족(이자겸의 난 1126, 묘청 서경천도 1135) → **무신정권**(최충헌 교정도감, 최우 정방·삼별초) → 대몽항쟁(강화도 천도·팔만대장경·삼별초 항쟁) → 원 간섭기(정동행성·권문세족) → **공민왕**(쌍성총관부 수복·정동행성 이문소 폐지·전민변정도감) → 신진사대부 성장.

**대외관계(매칭):** 거란 = **서희**(강동 6주)·**강감찬**(귀주대첩) / 여진 = **윤관**(별무반·동북 9성) / 몽골 = 대몽항쟁.

**문화재:** 팔만대장경(몽골 격퇴·해인사 현존) vs 초조대장경(거란 격퇴·소실), **직지심체요절**(최고 금속활자본·프랑스 소장), 의천(천태종·교관겸수) vs 지눌(조계종·정혜쌍수), 삼국사기(김부식·기전체) vs 삼국유사(일연·단군신화 수록), 상감청자, 주심포 건축.

## ④ 조선 전기 ★★★

**왕별 업적:** 태조(건국·정도전) → 태종(6조 직계제·호패법·사병 혁파) → **세종**(집현전·훈민정음·의정부 서사제·칠정산·농사직설·4군 6진·쓰시마 정벌) → 세조(6조 직계제 부활·직전법·경국대전 편찬 시작) → **성종**(**경국대전 완성·반포**·홍문관·동국통감·동국여지승람). 함정: 경국대전 = 세조 시작, **성종 완성**.

**통치 체제:** 의정부+6조, **3사**(사헌부=감찰·사간원=간쟁·홍문관=경연). 8도, **모든 군현에 수령 파견**(고려와 핵심 차이).

**사림과 4대 사화(순서 생명):** 무오(1498, 연산군, 조의제문) → 갑자(1504, 연산군, 폐비 윤씨) → **기묘(1519, 중종, 조광조·주초위왕)** → 을사(1545, 명종, 대윤·소윤). 두문자 "무–갑–기–을 / 연–연–중–명". 사림은 향촌(서원·향약)에서 재건.

**대외관계(사대교린):** 명=사대(조공·책봉), 여진=4군 6진, 일본=쓰시마 정벌→3포 개항→계해약조.

**문화:** 칠정산·농사직설·측우기(세종), 동국통감·국조오례의·악학궤범(성종), 혼일강리역대국도지도(태조). 분청사기·백자(전기) vs 청화백자(후기).

## ⑤ 조선 후기 ★★★

**양난·통치체제:** 임진왜란(이순신·권율·의병) → 광해군 중립외교 → 인조반정 → 정묘·병자호란(삼전도). **비변사** 최고 정무기구화. 5군영(훈련도감=삼수병)·속오군.

**수취 3법(최빈출):** **영정법**(전세, 1결 4두, 인조) / **대동법**(공납, 토지 결수 기준, 광해군 경기 시작→숙종 전국, **공인** 등장·선혜청) / **균역법**(군역, 군포 2필→1필, 영조, 결작 보충).

**정치 변동:** 붕당(예송 논쟁) → 환국(숙종) → 탕평 → 세도. **영조**(탕평비·균역법·속대전·청계천) vs **정조**(규장각·초계문신제·장용영·수원 화성·신해통공·대전통편). 세도정치(삼정 문란 → 홍경래의 난 1811·임술 농민 봉기 1862 → 삼정이정청).

**경제·사회:** 모내기(이앙법)·광작·장시·상평통보, 송상·만상·내상·경강상인. 공명첩·납속책으로 신분 동요, 공노비 해방(1801).

**실학:** 중농(유형원 균전론·이익 한전론·**정약용** 여전론·목민심서·거중기) / 중상(유수원·홍대용 지전설·**박지원** 열하일기·양반전·**박제가** 북학의·소비 권장) / 국학(안정복 동사강목·유득공 발해고·김정호 대동여지도·허준 동의보감).

**문화·사상:** 판소리·탈춤·한글소설, 진경산수화(정선) / 풍속화(김홍도·신윤복), 천주교(서학), **동학**(최제우, 1860, 인내천).

## ⑥ 근대 (개항기) ★★★

**흥선대원군:** 경복궁 중건(당백전·원납전)·서원 철폐·**호포제**·사창제·대전회통, 통상수교 거부(병인양요 1866 프랑스·신미양요 1871 미국·**척화비**).

**개항·불평등 조약:** **강화도조약(1876)**(운요호, 최초 근대적·불평등, 해안측량권·치외법권) → 조미수호통상조약(1882, 서양 최초, 최혜국·거중조정).

**임오·갑신:** 임오군란(1882, 구식군인) → 제물포조약·조청상민수륙무역장정 / 갑신정변(1884, 급진개화파, 14개조, 3일 천하) → 한성조약·톈진조약.

**동학농민운동(1894, 순서):** 고부민란 → 1차 봉기(반봉건) → 전주화약·**집강소**(폐정개혁) → 2차 봉기(반외세) → **우금치 패배** → 전봉준 체포.

**갑오/을미개혁(반드시 구분):** 1차 갑오(**군국기무처**, 신분제·과거제 폐지) → 2차 갑오(홍범 14조) → **을미개혁**(을미사변 후, **단발령·태양력**·건양) → 을미의병.

**독립협회·대한제국:** 독립협회(만민·관민공동회·헌의 6조) / 대한제국(1897, 광무개혁=**구본신참**, 대한국 국제, 양전·**지계** 발급).

**국권 피탈(조약 순서):** 한일의정서 → 1차 한일협약 → **을사늑약(1905, 외교권·통감부)** → 정미7조약(군대 해산) → 기유각서 → 한일병합(1910). 저항: 헤이그 특사(1907).

**의병·구국운동:** 을미(유생)·을사(최익현·신돌석)·정미(13도 창의군·서울 진공). 안중근(1909 이토 사살). **신민회**(안창호·양기탁, 공화정, 105인 사건). 국채보상운동(1907, 대구, 대한매일신보).

## ⑦ 일제강점기 (1910~1945)

**통치 3시기(등뼈):**
- **무단통치(1910s):** 헌병경찰제, **토지조사사업**(기한부 신고제·토지 약탈), **회사령**(허가제).
- **문화통치(1920s):** 3·1운동 후, 보통경찰제(실제 경찰 증가), **산미증식계획**(쌀 반출→조선인 소비 감소), 치안유지법(1925).
- **민족말살통치(1930~40s):** 황국신민화(신사참배·**창씨개명**), **국가총동원법(1938)**(징용·징병·위안부·공출).

**3·1운동 → 임정:** 3·1운동(1919) → 문화통치 전환 + **대한민국 임시정부**(상하이, 민주공화제, 연통제·교통국, 국민대표회의 1923 결렬, 충칭·한국광복군).

**무장투쟁(혼동 주의):** 봉오동·청산리(1920, 홍범도·김좌진=독립군 부대) ↔ **의열단**(1919 김원봉, 조선혁명선언) ↔ **한인애국단**(1931 김구, 이봉창·윤봉길) ↔ 한국광복군(1940, OSS 국내진공 준비).

**사회·국학:** 물산장려운동(조만식)·민립대학·형평운동(1923 진주)·6·10만세·광주학생항일(1929). **신간회**(1927~31, 정우회 선언, 좌우합작 최대 단체). 조선어학회(한글 맞춤법, 1942 사건), 신채호·박은식.

## [유네스코] 10 — 등재유산

- **Q. 세 범주를 한 마디로 구분하면?** → 세계유산=부동산(건물·유적·자연), 세계기록유산=동산 기록물(책·문서·활자), 인류무형문화유산=무형(의례·음악·기예).

## [문화재] 13 — 사진 단서 → 시대 양식

- **Q. 현존 最古·最大 석탑, 목탑 양식을 돌로 번안한 백제 탑은?** → 익산 미륵사지 석탑.

- **Q. 한쪽 다리 무릎 위, 손가락 뺨에 댄 사유 자세의 삼국 불상은?** → 금동미륵보살반가사유상(일본 고류사 영향).
- **Q. 부석사 무량수전·수덕사 대웅전·봉정사 극락전의 고려 건축 양식은?** → 주심포 양식(vs 조선 다포).
- **Q. 우리 자기의 시대 흐름 순서는?** → 고려청자(상감청자) → 분청사기(조선 초) → 백자.
- **Q. 8각·다층, 송 영향 고려 전기 대표 석탑은?** → 평창 월정사 8각9층석탑.
- **Q. (가)직지 (나)무구정광대다라니경 (다)팔만대장경 제작 순서는?** → (나) 통일신라 → (다) 고려(몽골 격퇴) → (가) 고려 말(최고 금속활자본).

## [시대 함정 모음] 01~08 대표 함정 카드 (교차 출제 단골)

- **[01 선사]** 비파형동검=청동기 / 세형동검=철기. 서옥제=고구려 / 민며느리제=옥저. 영고=부여 / 무천=동예. 소도·천군=삼한. 제천행사 없는 곳=옥저.
- **[02 고대]** 전성기 세기: 백(4)→고(5)→신(6). 녹읍 폐지=신문왕 / 정전=성덕왕 / 녹읍 부활=경덕왕. 금관가야 멸망=법흥왕 / 대가야 멸망=진흥왕.
- **[03 고려]** 초조대장경=거란·소실 / 팔만대장경=몽골·현존. 교관겸수=의천 / 정혜쌍수=지눌. 강동 6주=거란·서희 / 동북 9성=여진·윤관.
- **[04 조선전기]** 경국대전=세조 시작·**성종 완성**. 6조 직계제(태종·세조, 왕권↑) vs 의정부 서사제(세종, 재상권↑). 분청·백자=전기 / 청화백자=후기.
- **[05 조선후기]** 영정법=전세 / 대동법=공납(공인·선혜청) / 균역법=군역. 탕평비·균역법·속대전=영조 / 규장각·장용영·신해통공·수원화성=정조.
- **[06 근대]** 신분제·과거제 폐지=갑오개혁(군국기무처) / 단발령·태양력=을미개혁. 임오군란→제물포·조청장정 / 갑신정변→한성·톈진조약.
- **[07 일제]** 봉오동·청산리=독립군 부대(전투) / 의열단·한인애국단=개인 의거. 회사령=1910 허가제→1920 폐지. 산미증식=쌀 반출로 소비 감소.
- **[08 현대]** 발췌 개헌(1952)=직선제 / 사사오입(1954)=초대 대통령 중임 철폐. 직선제 개헌=6월 항쟁(1987). 최초 합의=7·4(1972) / 최초 정상회담=6·15(2000). 농지개혁: 남한=유상매수·유상분배.

> **마무리.** 1·2부로 "어느 시대에 무엇이 몇 문항"·"흐름"을 잡고, 3부 카드로 단골·함정을 확정 득점화한다. 핵심 영역(고대·고려·조선·근대)에서 함정만 안 틀려도 40문항(80%) 안정권에 도달한다.
